## Test
Make sure you are inside a **blank wordpress plugin directory** before proceeding.

Clone the repo.
```shell
$ git clone https://github.com/wpboots/boots.git boots
```
CD into the directory.
```shell
$ cd boots
```
Do a composer install.
```shell
$ composer install
```
Run the tests.
```shell
$ vendor/bin/phpunit tests
```
or
```shell
$ composer test
```
